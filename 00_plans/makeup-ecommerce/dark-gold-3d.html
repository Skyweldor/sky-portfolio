What changed — material and environment
/* ── Material ── */ new THREE.MeshPhysicalMaterial({ color: 0xd4a853 → 0x3d2200, /* dark brown base */ metalness: 1.0, /* unchanged */ roughness: 0.25 → 0.15, /* lower = sharper highlights */ envMapIntensity: 2.0 → 4.5, /* cranked — bright spots pop harder */ clearcoat: 0.5, clearcoatRoughness: 0.05, }) /* ── Environment ── */ envScene.background = 0x111111 → 0x010100 /* near-black ambient */ /* Key light: small + bright = tight specular */ keyLightPlane.scale = (10, 10) → (2.5, 2.5) /* smaller = harder highlight edge */ keyLightPlane.color = 0xfff8e0 /* warm white */ /* Fill light: barely visible */ fillLightPlane.color = 0xffe8cc → 0x0a0800 /* almost nothing */ /* ── Tone mapping ── */ renderer.toneMappingExposure = 1.2 → 0.9 /* pull exposure down */ /* The principle: dark base × dark environment × high envMapIntensity = only direct specular reflections survive as bright gold = everything else falls to deep brown/black */
Why this works
In PBR metallic rendering, the material's base color acts as the reflectance color — it tints all reflections. A bright base (#d4a853) means even dim environment reflections show up as visible gold. By dropping the base to #3d2200, dim reflections become near-invisible dark brown. Only high-intensity specular highlights — from the single key light — punch through as bright gold.

This is the exact same principle as the CSS fix: widen the dynamic range by lowering the floor, not raising the ceiling.
